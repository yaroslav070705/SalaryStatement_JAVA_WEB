1)Создание ного работника

(Если не находимся в главном меню, то нажиаем на 3 полоски слева вверху и нажимаем в выпавшем окне на кнопку "работники")
Главное меню -> Добавить работника -> заполнить данные в выпашем окне -> Добавить

2)Добавление сотридника в проект

Главное меню -> 3 полоски -> Проекты -> Ищем нужный проект в списке -> нажимаем на него -> Добавить работника -> вводим ФИО выбираем из выпавшего списка -> выбираем роль -> Добавить

3)Изменение размера выплаты за роль "X" в проекте "Y"

Главное меню -> 3 полоски -> Политики выплат -> По проектам -> выбираем проект из выпавшего списка -> Добавить -> указываем трубуемую роль и значение -> добавить

4)Получение проектов которые начались в определенный промежуток времени

Главное меню -> 3 полоски -> Проекты -> Фильтр -> Дата начала =... -> Применить

5) Получить все выплаты типа "X" для определенного сотрудника

Главное меню -> нажимаем на нужного сотрудника в списке -> Фильтр -> Выбор тип/дата -> применить
